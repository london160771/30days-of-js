# Day 01 — Array Methods Deep Dive

**Date:** <!-- 24/06/2026 -->
**Topic:** Core array methods in JavaScript

## What I practiced
- `map` — transform every item in an array
- `filter` — keep only items that match a condition
- `reduce` — fold an array into a single value
- `find` — get the first item that matches
- `some` / `every` — check conditions across the array
- `sort` — reorder items
- `flat` — collapse nested arrays

## How to run
Just open `index.html` in your browser. No build tools needed.
